# Чеклист ревью

## Корректность
- Публичные методы соответствуют заявленному описанию интерфейса (`kv.Store`, итераторы).
- Диапазон `Scan([start,end))` работает на крайних случаях (`nil`, пустой диапазон, совпадения).
- Поведение на отсутствующих ключах определено и проверено (`ErrNotFound`).

## Тесты
- Есть тесты на обновления ключей и удаления.
- Есть тесты на порядок итератора и стабильность границ.
- Тесты воспроизводимы (фиксированный seed, нет зависимости от времени).

## Ошибки и диагностика
- Ошибки не теряют контекст (оборачиваются/дополняются).
- Нет паник в библиотечном коде.

## Производительность
- Есть бенчмарки на критичные операции дня.
- Сборка не делает “очевидно лишних” аллокаций в горячих местах (по возможности).

## Документирование решений
- Есть короткая заметка (по шаблону `note_template.md`).
- Указано, какие варианты решений выбраны и почему.
